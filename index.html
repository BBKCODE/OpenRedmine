OpenRedmine

OpenRedmine is an android Redmine client.

App on Google Play

App on the store is master branch

Open source

This app is open source, you can contribute.

Download src as .zip Download src as .tar.gz
View on GitHub License

Features

  • Connection
    • Allow to connect UNSAFE SSL sites powered by transdroid
    • Allow to connect via Basic Authentication
    • View issues offline
    • Get API key from web site
  • Function
    • Filter (Status/Tracker/Category/Priority/Author/Assined)
    • Sort (IssueID/Created/Modified/... etc)
    • Show changelogs
    • Create or modify issue/timeentry
  • Translations (Languages)
    • English
    • Japanese (in_dow)

Target

  • Server
    • Redmine 1.2 later
    • API access key (Change from "My Account")
  • Client
    • Android 2.2 later

Version

Current commit is beta version.
Some feature is not complete or incorrect.
